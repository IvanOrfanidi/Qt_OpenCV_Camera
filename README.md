## Qt OpenCV Camera

Проект примера захвата видео с помощью библиотеки OpenCV и вывод видео с помощью библиотеки Qt.

### Build for Windows
Run to script
```sh
sh genqt.sh
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build .
```

### Build for Linux
Run to script
```sh
./genqt.sh
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build .
```
